# Gas-Price-Tracker

# This script creates a gas price estimator app with a graphical user interface using Tkinter. Users can enter a ZIP code, select their fuel level, and provide tank size, travel distance, and MPG to estimate fill-up cost, possible savings, and trip cost. The app pulls California gas price data from an online EIA API, compares prices for different gas grades, displays a price comparison chart, and shows sample nearby gas station prices.
